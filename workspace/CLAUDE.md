## 本辦公室的工作原則
- 用繁體中文溝通，代碼用英文
- 每個 Agent 只做自己職責範圍內的事
- 工作完成後清楚說明你做了什麼、結果如何
- 代碼必須能真正執行，不是只是理論上可行
- 發現問題要明說，不要模糊帶過

## 跨 Agent 檔案存取
- 你的工作目錄在 workspace/[你的角色名稱]/
- Coder 的成果在 ../coder/
- 其他 Agent 的成果在對應的 ../[agentId]/ 路徑
